## CS3431 Hospital Database Project
Margo Segui

This program is a hospital database management system built as a course project for **CS3431: Database Systems I** at Worcester Polytechnic Institue (WPI)

## Overview
This project implements a relational database for managing hospital data, including patients, doctors, staff, rooms, and medical records. It was developed in three phases, culminating in a Java application that connects to and queries an Oracle relational database

## Features
- Database schema design with normalized relational tables
- SQL scripts for table creation, data insertion, and querying
- Java application layer for interacting with the database via JDBC
- Support for common hospital operations, including:
  - patient admission and discharges
  - doctor and staff management
  - room assignments
  - medical record tracking

## Tech Stack
- **Language:** Java
- **IDE:** IntelliJ IDEA
- **Database:** Oracle SQL
- **Connectivity:** JDBC

## Course Context
This project was completed for **CS3431: Database Systems I** at WPI. It covers core database concepts including:
- Entity-Relationship Diagramming (ERD)
- Relational schema design and normalization
- SQL, DDL, and DML
- Application-level database access via JDBC
